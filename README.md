# HikeWay
An Android hiking app.

---

## Todo - Planned features
1. Interactive map and routes
The application provides an interactive map of the area with marked hiking routes. Each route contains basic information, such as name and short description, length, approximate time of passage, difficulty level, elevation gain, type of terrain.
The user can view routes on the map, open the detailed route page, filter them by difficulty, length, approximate time of passage, distance from the current GPS location or a selected point on the map.
* Hiking planner
The user can plan a hike. For a planned hike, weather warnings for the planned date, recommendations for starting time and return time taking into account daylight are available. The user can add a checklist for equipment to the planned hike.

2. Real-time GPS tracking
During the trip, the application uses GPS to track the user's movement in real time, displays the current location on the map and builds a track.

3. Offline navigation
The application allows you to download maps and routes for use without an Internet connection. In offline mode, map viewing, GPS navigation, track recording, viewing downloaded routes are available.

4. Travel log
Each completed trip is automatically saved in the user's personal log. The trip record contains the GPS track, date and time, total distance, duration, elevation gain.

5. Custom routes
Users can create their own routes. After publishing, the route becomes available to other users.

6. Reference information
Users can leave tags with interesting places on the map. Photos and comments can be attached to the tag.

7. Recommendation system
The application offers automatically generated route recommendations based on the desired difficulty, length, location, terrain type, previous user activities, and popularity of routes among other users. Recommendations are formed based on existing routes in the system.

8. Weather information
The application provides information about weather conditions on the route, including warnings about adverse conditions.

9. Social component
Users can rate routes, add photos to their routes and recorded trips, and view other users' published trips.

9.1 Gamification
There is a set of achievements that users can complete (first hike, 100 km, highest elevation gain).

10. Safety
SOS button (sending coordinates to a trusted contact).

11. Administration
Application administrators can moderate any user content, i.e. perform CRUD with users, routes, achievements, map markers.
